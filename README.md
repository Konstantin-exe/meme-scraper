# MeMe - Scraper

Fetching img from an given URL and saving them into the dir of ./memes.

## Installation

Install via yarn

```bash
yarn install
```

## Modules

npmjs node-fetch

```bash
yarn add node-fetch
```

npmjs request

```bash
yarn add request
```

## Usage

Simply run the programm, it will automatically create the dir ./memes and fetch the first 10 img from the given url.
